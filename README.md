
# Boas vindas ao repositório do projeto CookMaster!


## Instalação do projeto localmente

1. Abra o terminal e crie um diretório no local de sua preferência com o comando **mkdir**:
```javascript
  mkdir projetos
```

2. Entre no diretório que acabou de criar e depois clone o projeto:
```javascript
  cd projetos
  git clone git@github.com:LucasCAndre/CookMaster.git
```

3. Entre no diretorio raiz do projeto e Instale as dependencias do projeto:
```javascript
  cd CookMaster
  npm install
```
4. Em seguida rode o programa:
```javascript
  npm start
```
A conexão com o banco de dados é feita atravez do arquivo models/connection.js utilizando o mongodb, porém o banco de dados pode ser alterado de acordo com a preferencia alterando apenas esse arquivo.
